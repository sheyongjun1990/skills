# HTML 文件分类器

自动根据 HTML 文件的标题和内容，将其分类到汽车、星座、教育、娱乐等预设类别。

## 分类逻辑

- **汽车**：匹配关键词包括"汽车"、"宝马"、"奔驰"、"电动车"、"驾照"、"发动机"等
- **星座**：匹配关键词包括"星座"、"运势"、"白羊座"、"金牛座"、"十二星座"等
- **教育**：匹配关键词包括"教育"、"学校"、"大学"、"考试"、"高考"、"课程"等
- **娱乐**：匹配关键词包括"娱乐"、"明星"、"电影"、"电视剧"、"音乐"、"游戏"等

## 使用方法

### 直接运行

```bash
python scripts/classify_html.py
```

### 指定目录

```bash
python scripts/classify_html.py --dir /path/to/html/files
```

### 试运行模式（不移动文件，只显示分类结果）

```bash
python scripts/classify_html.py --dry-run
# 或
python scripts/classify_html.py -n
```

## 工作流程

1. 扫描当前目录下的所有 `.html` 和 `.htm` 文件
2. 提取每个文件的 HTML 标题 (`<title>`) 和正文内容
3. 基于关键词匹配算法计算每个类别的得分
4. 将文件移动到得分最高的对应类别文件夹
5. 输出分类统计报告

## 目录结构

执行后会自动创建以下文件夹（如果不存在）：

```
当前目录/
├── 汽车/          # 汽车相关 HTML 文件
├── 星座/          # 星座运势相关 HTML 文件
├── 教育/          # 教育学习相关 HTML 文件
├── 娱乐/          # 娱乐影视相关 HTML 文件
└── [原始文件]     # 未分类的文件保留在原位置
```

## 关键词权重

- 标题匹配：权重为 3
- 正文匹配：权重为 1

这意味着标题中出现的关键词对分类结果影响更大。

## 处理文件名冲突

如果目标文件夹中已存在同名文件，脚本会自动重命名（添加序号后缀）。

## 修改分类关键词

如需修改关键词，编辑 `scripts/classify_html.py` 文件中的 `CATEGORY_KEYWORDS` 字典。
