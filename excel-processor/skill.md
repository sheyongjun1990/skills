# Excel处理器技能 (excel-processor)

## 技能描述
这个技能用于读取Excel文件，筛选指定列的状态，将该列的值转换为JSON数组，并保存为文本文件。

## 功能特性
- 读取Excel文件（支持.xlsx, .xls格式）
- 筛选指定列中满足特定状态的行
- 将筛选后的值转换为JSON数组
- 自动生成输出文件名（原Excel名称+列名称+状态+时间.txt）
- 支持批量处理多个文件

## 使用方法

### 基本用法
```
处理Excel文件：读取[文件路径]，筛选[列名]列中状态为[状态值]的数据，转换为JSON数组并保存
```

### 参数说明
- **文件路径**: Excel文件的完整路径或相对路径
- **列名**: 要筛选的列名称（支持中文和英文）
- **状态值**: 要筛选的状态值（可以是文本、数字或布尔值）

### 示例用法

#### 示例1：基本筛选
```
处理Excel文件：读取D:\销售数据.xlsx，筛选"订单状态"列中状态为"已完成"的数据，转换为JSON数组并保存
```

#### 示例2：多条件处理
```
处理Excel文件：读取./用户数据.xlsx，筛选"status"列中状态为"active"的数据，转换为JSON数组并保存
```

#### 示例3：批量处理
```
批量处理Excel文件：读取文件夹D:\数据中的所有Excel文件，筛选"状态"列中状态为"通过"的数据，转换为JSON数组并保存
```

## 输出文件格式
- **文件名**: `原文件名_列名_状态_YYYYMMDD_HHMMSS.txt`
- **文件内容**: JSON数组格式，每行一个JSON对象

### 输出示例
假设原文件名为`销售数据.xlsx`，筛选列名为`订单状态`，状态值为`已完成`，输出文件名可能是：
```
销售数据_订单状态_已完成_20241229_143025.txt
```

文件内容格式：
```json
[
  {"订单ID": "001", "订单状态": "已完成", "金额": 100},
  {"订单ID": "002", "订单状态": "已完成", "金额": 200},
  {"订单ID": "003", "订单状态": "已完成", "金额": 150}
]
```

## 使用提示模板
```
处理Excel文件：读取{file_path}，筛选{column}列中状态为{status}的数据，转换为JSON数组并保存
批量处理Excel文件：读取{folder_path}中的所有Excel文件，筛选{column}列中状态为{status}的数据，转换为JSON数组并保存
```
